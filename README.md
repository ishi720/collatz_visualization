# 概要

### コラッツの問題

コラッツの問題とは、数論の未解決問題のひとつである。

自然数nに対して以下の処理を繰り返すと、どんな数でも必ず1に到達するといわれている。

- nが偶数の場合は、nを2で除算
- nが奇数の場合は、nに3を掛けて1を加算

### 例

```
6 → 3 → 10 → 5 → 16 → 8 → 4 → 2 → 1
```
